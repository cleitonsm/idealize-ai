# Frontend

Aplicacao Angular do Idealize AI.

Organizacao prevista:

- `core`: servicos globais, configuracoes, interceptors, guards e layout.
- `shared`: componentes, diretivas, pipes e modelos reutilizaveis.
- `features`: modulos funcionais da experiencia, como projetos, descoberta guiada, artefatos, diagramas e revisao.
